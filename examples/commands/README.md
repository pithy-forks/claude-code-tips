# example commands

copy any of these to `.claude/commands/` in your project, then invoke with `/<name>`.

| command | what it does |
|---|---|
| [/sweep](./sweep.md) | interactive dead code cleanup with confirmation |
| [/quicktest](./quicktest.md) | find and run tests for current file |

these are generic commands that work in any project. the commands i actually use daily (`/mine`, `/ship`, `/improve`) live in `.claude/commands/`.
